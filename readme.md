# if(kakao) 공식 컨퍼런스 사이트 클론 프로젝트

## 프로젝트 한 줄 요약
카카오 공식 컨퍼런스 **if(kakao)** 사이트를 HTML, CSS, JavaScript로 클론 구현한 개인 프로젝트로,  
PC·모바일 환경에 맞춘 **반응형 레이아웃과 UI 흐름**을 재현했습니다.

## 진행 기간
- 개인 프로젝트  
- 2025.11.12 ~ 2025.11.14 (2일)

## 주요 기능
- CSS 미디어 쿼리를 활용한 반응형 레이아웃 구현  
- 해상도별 Breakpoints 설정을 통한 디바이스 최적화  
- PC / 모바일 환경에 따른 UI 구조 변경  

## 기술 스택
- HTML5  
- CSS3  
- JavaScript (ES6+)

## 배운 점

### 반응형 레이아웃 설계 경험
화면 크기에 따라 레이아웃과 콘텐츠 배치가 달라지도록 구성하며,  
디바이스별로 다른 사용자 경험을 고려한 화면 설계를 경험할 수 있었습니다.

### 모바일 우선 UI 구성 이해
모바일 환경에서의 메뉴 구조와 콘텐츠 흐름을 먼저 고려해 설계하며,  
작은 화면에서 정보 전달의 중요성을 체감할 수 있었습니다.

### 해상도별 UI 테스트 경험
PC, 모바일 해상도에서 직접 테스트하며,  
레이아웃이 깨지는 지점을 확인하고 수정하는 경험을 할 수 있었습니다.

## 프로젝트 목적
- 반응형 웹 레이아웃 구현 능력 향상  
- 실제 서비스 사이트 UI 구조 분석 및 재현  
- 디바이스별 사용자 경험을 고려한 프론트엔드 설계 경험 축적
