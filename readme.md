# FlexK

## 프로젝트 한 줄 요약

카카오 공식 컨퍼런스 사이트 if(kakao)를 HTML, CSS, JavaScript로 클론 구현한 프로젝트로,  
PC·모바일 환경에 맞춰 반응형 레이아웃과 기본적인 UI 흐름을 재현한 개인 프로젝트입니다.

## 주요 기능

- CSS 미디어 쿼리를 활용한 반응형 레이아웃 구현  
- 화면 크기에 따른 레이아웃 변화 및 디바이스 환경 대응

## 기술 스택

---

## 배운 점 / 인사이트

### 반응형 레이아웃 경험

화면 크기에 따라 레이아웃과 콘텐츠 배치가 달라지도록 구성하며,  
디바이스 환경에 따른 사용자 경험 차이를 고려해보는 계기가 되었습니다.

### 모바일 우선 설계 경험

모바일 환경에서의 메뉴 구조와 콘텐츠 흐름을 먼저 고민하며,  
작은 화면에서 정보 전달의 우선순위가 중요하다는 점을 느꼈습니다.

### 테스트 기반 레이아웃 개선 경험

PC와 모바일 환경에서 직접 테스트하며 레이아웃이 깨지는 지점을 확인하고 수정하는 과정을 통해  
반응형 구현 경험을 쌓을 수 있었습니다.
