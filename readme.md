# if(kakao) Clone Project

## 한 줄 요약
카카오의 공식 컨퍼런스 사이트 if(kakao) 페이지를 HTML, CSS, JavaScript로 구현한 클론 프로젝트로, 메인 비주얼과 슬라이드·탭 UI 등 인터랙션 중심의 화면 구성을 재현했습니다.

---

## 진행 기간
- 개인 프로젝트  
- 2025.11.12 ~ 2025.11.19 (7일)

---

## 주요 기능

### 메인 비주얼 섹션 구현
- 비디오 배경과 텍스트 오버레이를 활용해 원본 사이트의 첫 화면 구성을 재현

### 슬라이드 UI 구현
- 이미지 슬라이드와 이전/다음 버튼, 인디케이터를 직접 구현해 콘텐츠 탐색 기능 제공

### 무한 슬라이드 및 캐러셀
- JavaScript를 활용해 콘텐츠가 자연스럽게 순환되는 무한 슬라이드 구조 구현

### 탭 & 아코디언 UI
- FAQ 영역에서 탭 전환 및 질문 클릭 시 열리는 아코디언 UI 구현

### 모바일 메뉴 인터랙션
- 모바일 환경을 고려한 햄버거 메뉴 및 팝업 네비게이션 구현

---

## 배운 점

- **순수 JavaScript로 UI 인터랙션 구현 경험**  
  라이브러리 없이 슬라이드, 탭, 아코디언 기능을 직접 구현하며 DOM 요소를 다루는 방법과 이벤트 처리 흐름을 익힐 수 있었습니다.

- **실제 서비스 화면 구조 분석 경험**  
  공식 사이트의 화면 구성과 콘텐츠 배치를 분석하며, 섹션별로 화면이 어떻게 구성되는지 이해할 수 있었습니다.

- **마크업 구조의 중요성 인식**  
  화면이 복잡할수록 HTML 구조를 어떻게 잡느냐가 CSS와 JavaScript 구현에 큰 영향을 준다는 점을 알게 되었습니다.
